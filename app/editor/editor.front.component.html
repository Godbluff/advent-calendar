Calendar
When creating a new calendar, all you have to do is type in the name of the calendar and create a password for it. This is all you need to enter the editor.

There is no recovery for the password at this time, so make sure you remember it.

Once you click 'Initiate Calendar' you will be taken to the main editor to set up your custom calendar.

Have fun.
